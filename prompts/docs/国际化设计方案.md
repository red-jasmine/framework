# 商品领域国际化设计方案

## 一、国际化核心原则

### 1. 参考的国际平台标准

| 平台 | 核心概念 | 特点 |
|------|---------|------|
| **Shopify** | Product + Variant | 最流行的国际电商SaaS平台 |
| **WooCommerce** | Product + Variation | WordPress生态，全球市场份额大 |
| **Magento** | Product + Simple/Configurable | 企业级电商平台 |
| **BigCommerce** | Product + Variant | 面向中大型企业 |

### 2. 国际化设计核心

```
✅ 使用国际通用术语（variant > sku）
✅ 支持多语言内容
✅ 支持多货币定价
✅ 标准化度量单位（SI国际单位制）
✅ 时区无关的时间存储
✅ 符合GDPR等国际法规
✅ SEO友好的元数据结构
✅ 国际税务支持
```

// 生成handle（URL友好标识）
Product::chunk(100, function ($products) {
    foreach ($products as $product) {
        $handle = Str::slug($product->title);
        $counter = 1;
        
        while (Product::where('handle', $handle)->exists()) {
            $handle = Str::slug($product->title) . '-' . $counter++;
        }
        
        $product->update(['handle' => $handle]);
    }
});

// 迁移单位数据
ProductSku::chunk(100, function ($skus) {

        $sku->update([
            'weight_unit' => 'kg',
            'dimension_unit' => 'm'
        ]);
    }
});
